## Basic usage

1.- Copy a file:

`$ node index.js package.json otherjson.json -c`

2.- Move a file:

`$ node index.js otherjson.json anothername.json`

3.- Copy a directory:

`$ node index.js ./ ../../../test -c`

4.- Move a directory

`$ node index.js ../../../test/ ../../../test2/`

## Run tests

`$ npm test`
